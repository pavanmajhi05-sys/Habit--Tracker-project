# Habit Tracker Web App

## 📌 Project Overview
The Habit Tracker Web App is a simple web application that helps users track their daily habits and maintain consistency.  
Users can add habits, mark them as completed, and monitor their progress over time.

This project was developed as part of a web development learning project and demonstrates basic frontend development using HTML, CSS, and JavaScript.

---

## 🎯 Objective
The objective of this project is to build a web-based tool that allows users to:

- Track daily habits
- Maintain productivity
- Monitor progress through simple visual indicators
- Encourage consistency and personal growth

---

## 🛠 Technologies Used

- **HTML** – Structure of the web application  
- **CSS** – Styling and layout  
- **JavaScript** – Application functionality  
- **Local Storage** – Storing habit data locally in the browser  

---

## ✨ Features

- Add new habits
- Mark habits as completed
- Track daily progress
- Simple and user-friendly interface
- Data stored locally using browser storage
- Responsive design for better usability
